## 详细测试步骤

### 1. 测试忘记密码功能

1. 打开浏览器开发者工具（F12）
2. 访问 http://localhost:4000/login
3. 确保在"登录"模式（不是注册）
4. 点击"忘记密码？"按钮
5. 应该在页面底部看到提示："密码重置链接已发送到您的手机"

**如果没有反应：**
- 检查浏览器控制台是否有 JavaScript 错误
- 尝试刷新页面
- 尝试无痕模式

### 2. 测试收货地址功能

#### 步骤 A：添加地址
1. 访问 http://localhost:4000/test-features
2. 点击"添加测试地址"按钮
3. 访问 http://localhost:4000/profile/addresses
4. 应该能看到刚添加的测试地址

#### 步骤 B：在结算页面使用地址
1. 访问 http://localhost:4000/shop
2. 点击任意商品的购物车图标添加到购物车
3. 访问 http://localhost:4000/cart
4. 点击"去结算"
5. 在结算页面应该能看到之前保存的地址
6. 可以选择地址或手动输入

**如果地址没有显示：**
- 打开浏览器控制台（F12）
- 在 Console 标签输入：`localStorage.getItem("user_addresses")`
- 检查是否有数据
- 如果没有数据，说明地址没有保存成功

### 3. 测试商城收藏功能

#### 步骤 A：添加收藏
1. 访问 http://localhost:4000/shop
2. 找到任意商品卡片
3. 点击商品图片右上角的爱心图标（❤️）
4. 应该看到提示："已收藏 XXX"
5. 爱心图标应该变成红色填充

#### 步骤 B：查看收藏
1. 访问 http://localhost:4000/profile/favorites
2. 应该能看到刚才收藏的商品

#### 步骤 C：取消收藏
1. 在商城页面再次点击红色爱心图标
2. 应该看到提示："已取消收藏 XXX"
3. 爱心图标变回灰色

**如果收藏功能不工作：**
- 打开浏览器控制台（F12）
- 查看是否有 JavaScript 错误
- 在 Console 输入：`localStorage.getItem("user_favorites")`
- 检查数据是否正确保存

## 常见问题

### 问题1：localStorage 被禁用

某些浏览器设置或扩展可能禁用 localStorage。

**解决方法：**
1. 在浏览器控制台输入：`typeof localStorage`
2. 应该返回 "object"
3. 如果返回 "undefined"，说明 localStorage 被禁用
4. 尝试在无痕模式下测试

### 问题2：浏览器缓存问题

旧的 JavaScript 文件可能被缓存。

**解决方法：**
1. 按 Ctrl+Shift+R（Windows）或 Cmd+Shift+R（Mac）强制刷新
2. 或者清除浏览器缓存
3. 或者在无痕模式下测试

### 问题3：React 状态未更新

页面状态可能没有正确更新。

**解决方法：**
1. 刷新页面
2. 检查浏览器控制台是否有错误
3. 重启开发服务器

### 问题4：点击事件被阻止

某些浏览器扩展可能阻止点击事件。

**解决方法：**
1. 禁用所有浏览器扩展
2. 在无痕模式下测试

## 调试技巧

### 1. 查看 localStorage 数据

在浏览器控制台输入：
```javascript
// 查看收藏
console.log(JSON.parse(localStorage.getItem("user_favorites") || "[]"));

// 查看地址
console.log(JSON.parse(localStorage.getItem("user_addresses") || "[]"));

// 查看用户信息
console.log(localStorage.getItem("user_token"));
console.log(JSON.parse(localStorage.getItem("user_info") || "{}"));
```

### 2. 手动添加测试数据

在浏览器控制台输入：
```javascript
// 添加测试收藏
localStorage.setItem("user_favorites", JSON.stringify([
  {
    id: 1,
    name: "测试商品",
    price: 99.99,
    image_url: "/placeholder.jpg",
    favorited_at: new Date().toISOString()
  }
]));

// 添加测试地址
localStorage.setItem("user_addresses", JSON.stringify([
  {
    id: "test-1",
    name: "张三",
    phone: "13800138000",
    province: "广东省",
    city: "广州市",
    district: "天河区",
    detail: "测试街道123号",
    isDefault: true
  }
]));

// 刷新页面
location.reload();
```

### 3. 清空所有数据

```javascript
localStorage.clear();
location.reload();
```
