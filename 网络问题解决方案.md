# Vercel 部署 - 网络问题解决方案

## 问题：npm 安装 Vercel CLI 失败

### 原因
网络连接问题，可能是：
1. 网络不稳定
2. npm 默认源在国外，访问慢
3. 防火墙或代理问题

---

## 解决方案

### 方案 A：使用国内镜像（推荐）

```bash
# 1. 切换到淘宝镜像
npm config set registry https://registry.npmmirror.com

# 2. 验证配置
npm config get registry

# 3. 重新安装 Vercel CLI
npm install -g vercel

# 4. 验证安装
vercel --version
```

### 方案 B：使用 npx（无需安装）

不安装 Vercel CLI，直接使用 npx：

```bash
# 直接运行 Vercel 命令
npx vercel

# 登录
npx vercel login

# 部署
npx vercel --prod
```

### 方案 C：使用 Vercel 网页部署（最简单）

**无需命令行，直接在网页上操作：**

#### 步骤 1：准备 GitHub 仓库

```bash
cd /home/xiaobai/code/biye/woganproject/anything/apps/web

# 初始化 Git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "Prepare for Vercel deployment"

# 在 GitHub 上创建仓库
# 访问 https://github.com/new
# 仓库名：wogan-project
# 设置为 Public

# 关联远程仓库（替换为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/wogan-project.git

# 推送代码
git branch -M main
git push -u origin main
```

#### 步骤 2：在 Vercel 网页部署

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击 "Sign Up"
   - 选择 "Continue with GitHub"（用 GitHub 账号登录）

2. **导入项目**
   - 登录后点击 "Add New..."
   - 选择 "Project"
   - 找到你的 `wogan-project` 仓库
   - 点击 "Import"

3. **配置项目**
   - Project Name: `wogan-project`
   - Framework Preset: 选择 "Other"
   - Root Directory: `./`
   - Build Command: `npm run build`
   - Output Directory: `build/client`
   - Install Command: `npm install`

4. **添加环境变量**
   - 点击 "Environment Variables"
   - 添加：
     ```
     Name: DATABASE_URL
     Value: (你的 Neon 数据库连接字符串)
     ```
   - 添加：
     ```
     Name: NODE_ENV
     Value: production
     ```

5. **部署**
   - 点击 "Deploy"
   - 等待 2-5 分钟
   - 部署完成后会显示你的网站链接

## 推荐流程（最简单）

### 第一步：创建 Neon 数据库

1. 访问 https://neon.tech
2. 用 GitHub 账号登录
3. 点击 "Create Project"
4. 项目名：`wogan-db`
5. 区域：`Asia Pacific (Singapore)`
6. 点击 "Create"
7. 复制连接字符串（类似）：
   ```
   postgresql://username:password@ep-xxx.region.aws.neon.tech/neondb?sslmode=require
   ```

### 第二步：初始化数据库

```bash
# 安装 PostgreSQL 客户端（如果还没有）
sudo apt-get install postgresql-client

# 连接到 Neon 数据库（替换为你的连接字符串）
psql "postgresql://username:password@ep-xxx.region.aws.neon.tech/neondb?sslmode=require"

# 在 psql 中执行
\i init_database.sql

# 退出
\q
```

### 第三步：推送代码到 GitHub

```bash
cd /home/xiaobai/code/biye/woganproject/anything/apps/web

# 如果还没有 Git 仓库
git init
git add .
git commit -m "Initial commit"

# 在 GitHub 创建仓库后
git remote add origin https://github.com/YOUR_USERNAME/wogan-project.git
git branch -M main
git push -u origin main
```

### 第四步：在 Vercel 网页部署

1. 访问 https://vercel.com
2. 用 GitHub 登录
3. 导入你的 `wogan-project` 仓库
4. 添加环境变量 `DATABASE_URL`
5. 点击 "Deploy"

🎉 完成！

## 完全离线部署方案

如果网络实在不行，可以使用本地服务器 + 内网穿透：

### 使用 Ngrok（临时方案）

```bash
# 1. 下载 Ngrok
wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.tgz
tar -xvzf ngrok-v3-stable-linux-amd64.tgz

# 2. 注册并获取 token
# 访问 https://ngrok.com

# 3. 启动应用
npm run dev

# 4. 在另一个终端启动 Ngrok
./ngrok http 4000

# 5. Ngrok 会给你一个公网地址
# 例如：https://abc123.ngrok.io
```
