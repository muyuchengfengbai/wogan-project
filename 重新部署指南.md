## 🎯 部署步骤（跟着做）

### 1️⃣ 删除旧项目（如果存在）

1. 进入 Vercel 控制台：https://vercel.com/dashboard
2. 找到你的项目，点击进入
3. 点击顶部 **Settings**
4. 滚动到最底部，点击 **Delete Project**
5. 输入项目名称确认删除

---

### 2️⃣ 创建新项目

1. 回到 Vercel 首页：https://vercel.com/dashboard
2. 点击 **Add New... → Project**
3. 找到 **muyuchengfengbai/wogan-project**
4. 点击 **Import**

---

### 3️⃣ 配置项目（重要！）

#### ✅ Root Directory（项目根目录）

```
apps/web
```

**⚠️ 注意：不要选择根目录，要选择 apps/web！**

---

#### ✅ Framework Preset（框架预设）

选择：**Other**（或者保持默认）

---

#### ✅ Build and Output Settings（构建设置）

**Build Command（构建命令）：**
```bash
npm install --legacy-peer-deps && npm run build
```

**Output Directory（输出目录）：**
```
build/client
```

**Install Command（安装命令）：**
```bash
npm install --legacy-peer-deps
```

---

#### ✅ Environment Variables（环境变量）

点击 **Add** 添加以下变量：

**变量名：**
```
DATABASE_URL
```

**变量值（完整复制）：**
```
postgresql://neondb_owner:npg_lhbKNrZLGs27@ep-long-bar-aovy3qu6-pooler.c-2.ap-southeast-1.aws.neon.tech/neondb?sslmode=require
```

**⚠️ 重要：确保完整复制，不要有空格或换行！**

---

### 4️⃣ 开始部署

1. 检查所有配置是否正确
2. 点击 **Deploy** 按钮
3. 等待 2-3 分钟

---

### 5️⃣ 查看部署状态

- 📦 **Installing dependencies**（安装依赖）
