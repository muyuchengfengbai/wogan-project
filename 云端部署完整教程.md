## 方案一：Vercel + Neon（推荐，最简单）

### 为什么选择这个方案？
- ✅ **完全免费**（个人项目足够用）
- ✅ **部署超简单**（5分钟完成）
- ✅ **自动HTTPS**
- ✅ **全球CDN加速**
- ✅ **自动CI/CD**（推送代码自动部署）

### 步骤 1：准备代码

#### 1.1 修改数据库配置

编辑 `src/app/api/utils/sql.js`：

```javascript
import { Pool } from 'pg';

// 使用环境变量
const pool = new Pool({
  connectionString: process.env.DATABASE_URL,
  ssl: process.env.NODE_ENV === 'production' ? { rejectUnauthorized: false } : false
});

export async function query(text, params) {
  const client = await pool.connect();
  try {
    const result = await client.query(text, params);
    return result;
  } finally {
    client.release();
  }
}
```

#### 1.2 创建 `.env.example` 文件

```bash
# 数据库连接
DATABASE_URL=postgresql://user:password@host:5432/database

# Node 环境
NODE_ENV=production
```

#### 1.3 修改 `package.json`

确保有以下脚本：

```json
{
  "scripts": {
    "dev": "react-router dev",
    "build": "react-router build",
    "start": "react-router-serve ./build/server/index.js",
    "typecheck": "react-router typegen && tsc"
  }
}
```

#### 1.4 创建 `vercel.json`

```json
{
  "buildCommand": "npm run build",
  "outputDirectory": "build/client",
  "devCommand": "npm run dev",
  "installCommand": "npm install",
  "framework": null,
  "rewrites": [
    {
      "source": "/api/:path*",
      "destination": "/api/:path*"
    },
    {
      "source": "/(.*)",
      "destination": "/"
    }
  ]
}
```

### 步骤 2：创建 Neon 数据库

#### 2.1 注册 Neon 账号

1. 访问 https://neon.tech
2. 点击 "Sign Up" 注册（可以用 GitHub 账号登录）
3. 免费版提供：
   - 0.5 GB 存储
   - 无限查询
   - 自动备份

#### 2.2 创建数据库

1. 登录后点击 "Create Project"
2. 项目名称：`wogan-project`
3. 区域选择：`Asia Pacific (Singapore)` 或 `US East`
4. PostgreSQL 版本：选择最新版
5. 点击 "Create Project"

#### 2.3 获取连接字符串

1. 在项目页面找到 "Connection String"
2. 复制类似这样的字符串：
   ```
   postgresql://username:password@ep-xxx.region.aws.neon.tech/neondb?sslmode=require
   ```
3. 保存这个字符串，稍后会用到

#### 2.4 初始化数据库

1. 在 Neon 控制台点击 "SQL Editor"
2. 粘贴你的 `init_database.sql` 内容
3. 点击 "Run" 执行
4. 或者使用命令行：

```bash
# 安装 psql 客户端
# Ubuntu/Debian
sudo apt-get install postgresql-client

# macOS
brew install postgresql

# 连接到 Neon 数据库
psql "postgresql://username:password@ep-xxx.region.aws.neon.tech/neondb?sslmode=require"

# 执行初始化脚本
\i init_database.sql
```

### 步骤 3：部署到 Vercel

#### 3.1 安装 Vercel CLI

```bash
npm install -g vercel
```

#### 3.2 登录 Vercel

```bash
vercel login
```

选择登录方式（GitHub、GitLab、Email 等）

#### 3.3 部署项目

```bash
cd /home/xiaobai/code/biye/woganproject/anything/apps/web

# 首次部署
vercel

# 按照提示操作：
# - Set up and deploy? Yes
# - Which scope? 选择你的账号
# - Link to existing project? No
# - Project name? wogan-project
# - In which directory is your code located? ./
# - Want to override the settings? No
```

#### 3.4 配置环境变量

```bash
# 添加数据库连接字符串
vercel env add DATABASE_URL

# 粘贴你的 Neon 连接字符串
# 选择环境：Production, Preview, Development (全选)

# 添加 NODE_ENV
vercel env add NODE_ENV
# 输入：production
```

#### 3.5 重新部署

```bash
vercel --prod
```

### 步骤 4：访问你的网站

部署完成后，Vercel 会给你一个链接，类似：
```
https://wogan-project.vercel.app
```

🎉 恭喜！你的网站已经上线了！

### 步骤 5：绑定自定义域名（可选）

#### 5.1 在 Vercel 控制台

1. 进入项目设置
2. 点击 "Domains"
3. 输入你的域名（例如：wogan.yourdomain.com）
4. 点击 "Add"

#### 5.2 配置 DNS

在你的域名提供商（阿里云、腾讯云等）添加 CNAME 记录：

```
类型: CNAME
主机记录: wogan
记录值: cname.vercel-dns.com
TTL: 600
```

等待 DNS 生效（通常 5-10 分钟）

## 方案二：Railway（一站式部署）

### 为什么选择 Railway？
- ✅ 数据库和应用一起部署
- ✅ 配置简单
- ✅ 自动备份
- ⚠️ 免费额度：$5/月（够用）

### 步骤 1：注册 Railway

1. 访问 https://railway.app
2. 用 GitHub 账号登录
3. 验证邮箱

### 步骤 2：创建项目

1. 点击 "New Project"
2. 选择 "Deploy from GitHub repo"
3. 授权 Railway 访问你的 GitHub
4. 选择你的项目仓库

### 步骤 3：添加 PostgreSQL

1. 在项目页面点击 "New"
2. 选择 "Database" → "PostgreSQL"
3. Railway 会自动创建数据库
4. 自动设置 `DATABASE_URL` 环境变量

### 步骤 4：配置构建

在项目根目录创建 `railway.json`：

```json
{
  "build": {
    "builder": "NIXPACKS",
    "buildCommand": "npm install && npm run build"
  },
  "deploy": {
    "startCommand": "npm start",
    "restartPolicyType": "ON_FAILURE",
    "restartPolicyMaxRetries": 10
  }
}
```

### 步骤 5：初始化数据库

1. 在 Railway 控制台找到 PostgreSQL 服务
2. 点击 "Connect"
3. 复制连接命令
4. 在本地执行：

```bash
psql "postgresql://postgres:xxx@xxx.railway.app:5432/railway"
\i init_database.sql
```

### 步骤 6：部署

推送代码到 GitHub，Railway 会自动部署：

```bash
git add .
git commit -m "Deploy to Railway"
git push
```

部署完成后，Railway 会给你一个链接：
```
https://wogan-project-production.up.railway.app
```

## 方案三：阿里云/腾讯云（国内访问快）

### 为什么选择国内云？
- ✅ 国内访问速度快
- ✅ 稳定可靠
- ⚠️ 需要备案（个人备案约 20 天）
- ⚠️ 费用较高（约 ¥100-300/月）

### 推荐配置

**服务器配置**：
- CPU: 2核
- 内存: 4GB
- 硬盘: 40GB SSD
- 带宽: 5Mbps
- 系统: Ubuntu 22.04 LTS

**费用估算**：
- 阿里云轻量应用服务器：¥108/年
- 腾讯云轻量应用服务器：¥112/年

### 步骤 1：购买服务器

#### 阿里云
1. 访问 https://www.aliyun.com
2. 选择"轻量应用服务器"
3. 选择配置和地域
4. 购买并完成实名认证

#### 腾讯云
1. 访问 https://cloud.tencent.com
2. 选择"轻量应用服务器"
3. 选择配置和地域
4. 购买并完成实名认证

### 步骤 2：连接服务器

```bash
# 使用 SSH 连接
ssh root@your-server-ip

# 首次登录会要求修改密码
```

### 步骤 3：安装环境

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 安装 PostgreSQL
apt install -y postgresql postgresql-contrib

# 安装 Nginx
apt install -y nginx

# 安装 PM2（进程管理器）
npm install -g pm2
```

### 步骤 4：配置 PostgreSQL

```bash
# 切换到 postgres 用户
sudo -u postgres psql

# 创建数据库和用户
CREATE DATABASE wogan_db;
CREATE USER wogan_user WITH PASSWORD 'your_strong_password';
GRANT ALL PRIVILEGES ON DATABASE wogan_db TO wogan_user;
\q

# 允许远程连接（可选）
nano /etc/postgresql/14/main/postgresql.conf
# 修改：listen_addresses = '*'

nano /etc/postgresql/14/main/pg_hba.conf
# 添加：host all all 0.0.0.0/0 md5

# 重启 PostgreSQL
systemctl restart postgresql
```

### 步骤 5：上传项目

```bash
# 在本地打包项目
cd /home/xiaobai/code/biye/woganproject/anything/apps/web
tar -czf wogan-project.tar.gz .

# 上传到服务器
scp wogan-project.tar.gz root@your-server-ip:/root/

# 在服务器上解压
ssh root@your-server-ip
mkdir -p /var/www/wogan
cd /var/www/wogan
tar -xzf ~/wogan-project.tar.gz

# 安装依赖
npm install --production

# 创建 .env 文件
nano .env
```

`.env` 内容：
```
DATABASE_URL=postgresql://wogan_user:your_strong_password@localhost:5432/wogan_db
NODE_ENV=production
PORT=4000
```

### 步骤 6：初始化数据库

```bash
# 导入数据
psql -U wogan_user -d wogan_db -f init_database.sql
```

### 步骤 7：配置 PM2

```bash
# 启动应用
pm2 start npm --name "wogan-app" -- start

# 设置开机自启
pm2 startup
pm2 save

# 查看状态
pm2 status
pm2 logs wogan-app
```

### 步骤 8：配置 Nginx

```bash
# 创建 Nginx 配置
nano /etc/nginx/sites-available/wogan
```

配置内容：
```nginx
server {
    listen 80;
    server_name your-domain.com;  # 替换为你的域名或 IP

    location / {
        proxy_pass http://localhost:4000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

```bash
# 启用配置
ln -s /etc/nginx/sites-available/wogan /etc/nginx/sites-enabled/
nginx -t
systemctl restart nginx
```

### 步骤 9：配置 HTTPS（可选但推荐）

```bash
# 安装 Certbot
apt install -y certbot python3-certbot-nginx

# 获取 SSL 证书
certbot --nginx -d your-domain.com

# 自动续期
certbot renew --dry-run
```

### 步骤 10：配置防火墙

```bash
# 允许 HTTP 和 HTTPS
ufw allow 80/tcp
ufw allow 443/tcp
ufw allow 22/tcp  # SSH
ufw enable
```

### 数据库备份

**Neon**:
自动备份，可在控制台恢复

**Railway**:
```bash
railway run pg_dump > backup.sql
```

**自建服务器**:
```bash
# 创建备份脚本
nano /root/backup-db.sh
```

```bash
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
pg_dump -U wogan_user wogan_db > /root/backups/wogan_db_$DATE.sql
# 保留最近 7 天的备份
find /root/backups -name "wogan_db_*.sql" -mtime +7 -delete
```

```bash
chmod +x /root/backup-db.sh
# 添加到 crontab（每天凌晨 2 点备份）
crontab -e
# 添加：0 2 * * * /root/backup-db.sh
```

## 常见问题

### 1. 部署后页面空白

**原因**：构建失败或路由配置错误

**解决**：
```bash
# 检查构建日志
vercel logs

# 本地测试生产构建
npm run build
npm start
```

### 2. 数据库连接失败

**原因**：环境变量未设置或连接字符串错误

**解决**：
```bash
# 检查环境变量
vercel env ls

# 测试数据库连接
psql "your-database-url"
```

### 3. API 请求 404

**原因**：路由配置问题

**解决**：
检查 `vercel.json` 的 rewrites 配置

### 4. 图片无法显示

**原因**：图片路径错误或未上传

**解决**：
- 使用绝对路径
- 或使用图床服务（如阿里云 OSS、七牛云）

### 5. 性能优化

**CDN 加速**：
- Vercel 自带全球 CDN
- 自建服务器可使用阿里云 CDN 或 Cloudflare

**数据库优化**：
```sql
-- 添加索引
CREATE INDEX idx_products_grade ON products(grade);
CREATE INDEX idx_orders_status ON orders(status);
CREATE INDEX idx_orders_created_at ON orders(created_at);
```
