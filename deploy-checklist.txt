Vercel + Neon 部署清单
======================

□ 步骤 1：配置 npm 镜像
  命令：npm config set registry https://registry.npmmirror.com

□ 步骤 2：修改数据库配置
  文件：src/app/api/utils/sql.js

□ 步骤 3：创建 Vercel 配置
  文件：vercel.json, .vercelignore

□ 步骤 4：初始化 Git
  命令：git init && git add . && git commit -m "Initial commit"

□ 步骤 5：创建 GitHub 仓库
  网址：https://github.com/new
  仓库名：wogan-project

□ 步骤 6：推送代码
  命令：git push -u origin main

□ 步骤 7：注册 Neon
  网址：https://neon.tech
  创建项目：wogan-db

□ 步骤 8：初始化数据库
  在 Neon SQL Editor 执行 init_database.sql

□ 步骤 9：注册 Vercel
  网址：https://vercel.com
  用 GitHub 登录

□ 步骤 10：导入项目
  选择 wogan-project 仓库

□ 步骤 11：配置环境变量
  DATABASE_URL = (Neon 连接字符串)
  NODE_ENV = production

□ 步骤 12：部署
  点击 Deploy 按钮

□ 步骤 13：测试访问
  访问：https://wogan-project.vercel.app

完成！🎉
