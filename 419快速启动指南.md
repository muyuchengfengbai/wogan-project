# 沃柑智慧产销一体化平台 - 快速启动指南

## 一、环境准备（首次运行）

### 1. 激活 Conda 环境
```bash
conda activate wogan_project
```

### 2. 进入项目目录
```bash
cd /home/xiaobai/code/biye/woganproject/anything/apps/web
```

### 3. 安装依赖（首次运行）
```bash
npm install --legacy-peer-deps
```

## 二、启动数据库

### 检查数据库状态
```bash
pg_ctl -D pgdata status
```

### 如果数据库未运行，启动数据库
```bash
pg_ctl -D pgdata start
```

### 验证数据库连接
```bash
psql -U xiaobai -d wogan_db -c "SELECT COUNT(*) FROM products;"
```

## 三、启动应用

### 启动开发服务器
```bash
npm run dev
```

**停止数据库**:
```bash

```

## 五、快速启动脚本

### 创建启动脚本（可选）
```bash
cat > start.sh << 'EOF'
#!/bin/bash
echo "=== 沃柑平台启动脚本 ==="

# 激活环境
source ~/miniconda3/etc/profile.d/conda.sh
conda activate wogan_project

# 进入项目目录
cd /home/xiaobai/code/biye/woganproject/anything/apps/web

# 检查并启动数据库
echo "检查数据库状态..."
if ! pg_ctl -D pgdata status > /dev/null 2>&1; then
    echo "启动数据库..."
    pg_ctl -D pgdata start
    sleep 2
else
    echo "数据库已运行"
fi

# 启动开发服务器
echo "启动开发服务器..."
echo "访问地址: http://localhost:4000"
npm run dev
EOF

chmod +x start.sh
```

### 使用启动脚本
```bash
./start.sh
```

## 六、故障排查

### 问题1: 端口被占用
```bash
# 查找占用 4000 端口的进程
lsof -i :4000

# 杀死进程（替换 PID）
kill -9 <PID>
```

### 问题2: 数据库连接失败
```bash
# 检查数据库是否运行
pg_ctl -D pgdata status

# 重启数据库
pg_ctl -D pgdata restart

# 检查连接配置
cat .env | grep DATABASE_URL
```

### 问题3: npm 依赖错误
```bash
# 清理缓存
rm -rf node_modules package-lock.json

# 重新安装
npm install --legacy-peer-deps
```

### 问题4: 页面空白或报错
```bash
# 查看浏览器控制台错误
# 按 F12 打开开发者工具

# 查看服务器日志
# 终端会显示详细错误信息
```
