# 解决 GitHub 403 错误

## 问题原因
- Token 权限不足
- Token 输入错误（复制时多了空格）
- Token 已过期
- 仓库不存在或无权限

---

## 🎯 解决方案（按顺序尝试）

### 方案 1：重新生成 Token（推荐）

#### 步骤 1：生成新 Token

1. **访问**：https://github.com/settings/tokens

2. **删除旧 Token**（如果有）

3. **点击**：`Generate new token` → `Generate new token (classic)`

4. **填写**：
   - Note: `wogan-deployment`
   - Expiration: `No expiration`（永不过期）
   - **勾选权限**（重要）：
     - ✅ `repo`（展开后全部勾选）
     - ✅ `workflow`
     - ✅ `write:packages`
     - ✅ `delete:packages`

5. **点击**：`Generate token`

6. **复制 Token**（格式：`ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`）

#### 步骤 2：清除旧凭据并重新推送

```bash
cd /home/xiaobai/code/biye/woganproject/anything/apps/web

# 清除 Git 凭据缓存
git credential-cache exit
git config --global --unset credential.helper

# 重新推送
git push -u origin main
```

**输入时注意**：
- Username: `muyuchengfengbai`
- Password: **完整的 Token**（确保没有多余空格）

### 方案 2：使用 Token 直接嵌入 URL

```bash
cd /home/xiaobai/code/biye/woganproject/anything/apps/web

# 将 YOUR_TOKEN 替换为你的实际 Token
git remote set-url origin https://YOUR_TOKEN@github.com/muyuchengfengbai/wogan-project.git

# 推送
git push -u origin main
```

**示例**：
```bash
# 如果你的 Token 是 ghp_abc123xyz
git remote set-url origin https://ghp_abc123xyz@github.com/muyuchengfengbai/wogan-project.git
```

### 方案 3：确认仓库是否存在

1. **访问**：https://github.com/muyuchengfengbai/wogan-project

2. **如果显示 404**，需要先创建仓库：
   - 访问：https://github.com/new
   - Repository name: `wogan-project`
   - 选择：`Public`
   - **不要**勾选 `Add a README file`
   - 点击：`Create repository`

3. **创建后再推送**：
   ```bash
   cd /home/xiaobai/code/biye/woganproject/anything/apps/web
   git push -u origin main
   ```

### 方案 4：使用 SSH（一劳永逸）

```bash
# 生成 SSH 密钥
ssh-keygen -t ed25519 -C "muyuchengfengbai@users.noreply.github.com"
# 按回车使用默认路径，按回车跳过密码

# 显示公钥
cat ~/.ssh/id_ed25519.pub
# 复制输出内容（以 ssh-ed25519 开头）
```

**添加到 GitHub**：
1. 访问：https://github.com/settings/keys
2. 点击：`New SSH key`
3. Title: `My Computer`
4. Key: 粘贴公钥
5. 点击：`Add SSH key`

**修改远程地址并推送**：
```bash
cd /home/xiaobai/code/biye/woganproject/anything/apps/web

# 修改为 SSH 地址
git remote set-url origin git@github.com:muyuchengfengbai/wogan-project.git

# 推送
git push -u origin main
```
