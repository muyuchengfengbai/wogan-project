# 数据库访问指南

## 数据库位置
**当前数据库运行在本地服务器上**，不是云端服务器。

## 数据库信息
- **数据库类型**: PostgreSQL
- **数据库名**: wogan_db
- **用户名**: xiaobai
- **端口**: 5432
- **连接地址**: localhost
- **数据目录**: /home/xiaobai/code/biye/woganproject/anything/apps/web/pgdata

## 访问方式

### 1. 命令行访问（推荐）
```bash
# 进入 conda 环境
conda activate wogan_project

# 连接数据库
psql -U xiaobai -d wogan_db

# 常用 SQL 命令
\dt                    # 查看所有表
\d products           # 查看 products 表结构
SELECT * FROM products LIMIT 5;  # 查询商品数据
SELECT * FROM orders;  # 查询订单数据
\q                     # 退出
```

### 2. 使用 pgAdmin（图形界面工具）
1. 下载安装 pgAdmin: https://www.pgadmin.org/download/
2. 创建新连接：
   - Host: localhost
   - Port: 5432
   - Database: wogan_db
   - Username: xiaobai
   - Password: (如果设置了密码)

### 3. 使用 DBeaver（通用数据库工具）
1. 下载安装 DBeaver: https://dbeaver.io/download/
2. 新建连接 → PostgreSQL
3. 填入上述连接信息

### 4. VS Code 插件
安装 PostgreSQL 插件：
- 插件名称: PostgreSQL (by Chris Kolkman)
- 配置连接信息后可直接在 VS Code 中查询

## 数据库表结构

### products（商品表）
- id: 商品ID
- name: 商品名称
- grade: 等级（精品果/一级果/普通果）
- price: 价格
- stock_kg: 库存（千克）
- origin: 产地
- image_url: 图片URL
- batch_no: 批次号
- sales_count: 销量

### orders（订单表）
- id: 订单ID
- order_no: 订单编号
- user_id: 用户ID
- total_amount: 总金额
- status: 状态（pending/paid/shipped/completed/cancelled）
- receiver_name: 收货人
- receiver_phone: 收货电话
- receiver_address: 收货地址

### order_items（订单明细表）
- id: 明细ID
- order_id: 订单ID
- product_id: 商品ID
- product_name: 商品名称
- price: 单价
- quantity: 数量
- subtotal: 小计

### traceability（溯源表）
- id: 溯源ID
- product_id: 商品ID
- batch_no: 批次号
- action_type: 操作类型（种植/施肥/采摘/分拣/包装）
- action_date: 操作日期
- location: 地点
- operator: 操作人
- details: 详情

### orchard_sensors（果园传感器表）
- id: 传感器ID
- sensor_name: 传感器名称
- temperature: 温度
- humidity: 湿度
- soil_moisture: 土壤湿度
- light_intensity: 光照强度
- recorded_at: 记录时间

## 数据库管理

### 启动数据库
```bash
conda activate wogan_project
cd /home/xiaobai/code/biye/woganproject/anything/apps/web
pg_ctl -D pgdata start
```

### 停止数据库
```bash
pg_ctl -D pgdata stop
```

### 查看数据库状态
```bash
pg_ctl -D pgdata status
```

### 备份数据库
```bash
pg_dump -U xiaobai wogan_db > backup_$(date +%Y%m%d).sql
```

### 恢复数据库
```bash
psql -U xiaobai -d wogan_db < backup_20240403.sql
```

## 迁移到服务器

如果将来需要部署到服务器，有两种方案：

### 方案1：使用云数据库服务
- 阿里云 RDS PostgreSQL
- 腾讯云 PostgreSQL
- AWS RDS PostgreSQL

### 方案2：自建服务器数据库
1. 在服务器上安装 PostgreSQL
2. 导出本地数据：`pg_dump -U xiaobai wogan_db > wogan_db.sql`
3. 上传到服务器并导入
4. 修改 .env 文件中的 DATABASE_URL

## 注意事项
1. 数据库文件存储在本地，重装系统前务必备份
2. 定期备份数据库，建议每天备份一次
3. 生产环境建议使用云数据库服务，更稳定可靠
4. 不要将数据库密码提交到 Git 仓库
